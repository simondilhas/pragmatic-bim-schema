    Class: Requirement¶

    Prescriptive requirement record (content_kind requirement). Not an Entity; may apply to one or more model entities. Domain is discriminated by concrete subclass (PerformanceRequirement, SpatialRequirement, etc.), not a separate slot.

    __

    • NOTE: this is an abstract class and should not be instantiated directly
